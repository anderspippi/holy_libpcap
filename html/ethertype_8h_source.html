23  * Ethernet types.
24  *
25  * We wrap the declarations with #ifdef, so that if a file includes
26  * <netinet/if_ether.h>, which may declare some of these, we don't
27  * get a bunch of complaints from the C compiler about redefinitions
28  * of these values.
29  *
30  * We declare all of them here so that no file has to include
31  * <netinet/if_ether.h> if all it needs are ETHERTYPE_ values.
32  */
